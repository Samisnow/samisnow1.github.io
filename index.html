Blaze

UX DESIGN, PROJECT MANAGMENT

On a team of developers, I designed the UX and UI of the ios Blaze app. Blaze is an app that helps artists put together colors to create a scheme. Many times artists find it difficult to find the perfect hue for their art piece. Blaze helps the artist define a color scheme to be used on a painting, from a one color focus. Then, the app dynamically outputs a set of color swatches the artist can use. In the beginning, the team and I defined the scope of the project, and then I utilized Omnigraffle to create the first wire frame of the main screen UI. Then prior to development I designed Photoshop mock ups and added specific pixel perfect UI icons via Illustrator.
